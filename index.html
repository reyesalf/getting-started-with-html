Important concepts

How the web works

The web is a collection of HTML documents. A browser requests to servers these documents using HTTP. A webpage is a text document written in a language called HTML. Web browsers read these documents, interpret them and display them. This video explains it in a simple manner.

How coding works

Coding happens when programmer write text in a language that a computer can understand. Computers are stupid and programmers need to write exactly the way that a computer will understand by using the correct syntax.

HTML

HTML documents contains text content (which represents "what you see"), markups (which represents "what it looks like"), references to other documents and links to other pages.


HTML attributes: tags and elements

HTML documents are made of HTML elements. When we write in HTML, the browsers interpret the type of each element using HTML tags.

A tag is always contained within angled brackets. Most tags have an opening tag and a closing tag. Some tags called "void" tags do not require a closing tag.

An element refers to everything within a set of opening and closing tags.

An attribute is a property of an HTML element. Elements can be inline or block. The difference between these elements is that block elements form an "invisible box" around the content inside of them.
